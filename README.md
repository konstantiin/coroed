## Features

- Fixed thread pool executor
- Stackfull coroutines
- Platform: x64
- Zombie-fibers
- Event
- Sleep

## Build & Run

Getting started.

```bash
make clean && bear -- make compile && make
```

Run precommit checks locally.

```bash
./ci/precommit.bash
```
